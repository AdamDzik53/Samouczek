1. Czym jest HTML?

HTML (HyperText Markup Language) jest językiem znaczników służącym do tworzenia struktury stron internetowych.
Za pomocą HTML określamy:

  • jakie elementy znajdują się na stronie,
  • kolejność elementów,
  • strukturę treści,
  • formularze,
  • tabele,
  • obrazy,
  • linki.

HTML nie odpowiada za wygląd strony. Jego zadaniem jest opisanie zawartości.
Można powiedzieć, że HTML jest szkieletem strony internetowej.

2. Budowa dokumentu HTML

Każda strona HTML posiada podobną strukturę. <!DOCTYPE html>
<html lang="pl">
<head>
<meta charset="UTF-8">
<title>Moja strona</title>
</head>
<body>
Zawartość strony
</body>
</html>

<!DOCTYPE html>
Informuje przeglądarkę, że dokument korzysta z HTML5.

<html>
Główny kontener całej strony.

<head>
Przechowuje informacje o stronie niewidoczne dla użytkownika. Znajdują się tutaj: tytuł strony, kodowanie znaków, połączenie z CSS, metadane.

<meta charset="UTF-8">
Najczęściej: Pozwala poprawnie wyświetlać polskie znaki.

<title>
Tytuł wyświetlany na karcie przeglądarki.
<title>Moja strona</title>

<body>
Zawiera wszystkie elementy widoczne dla użytkownika.

4. Nagłówki

HTML posiada sześć poziomów nagłówków.

<h1>Najważniejszy nagłówek</h1>
<h2>Rozdział</h2>
<h3>Podrozdział</h3>
<h4>Mniej ważny</h4>
<h5>Jeszcze mniej ważny</h5>
<h6>Najmniej ważny</h6>

Znaczenie:
Nagłówki tworzą strukturę dokumentu.

Typowa hierarchia:
H1 - Tytuł strony
H2 - Rozdział
H3 - Podrozdział
H4 - Szczegóły
H5 - Dodatkowe informacje
H6 - Najmniej ważne informacje

Na stronie zazwyczaj występuje tylko jeden nagłówek H1.

10. Formularze

Formularze umożliwiają użytkownikowi wprowadzanie danych na stronie internetowej.

Przykładowe zastosowania:

  • logowanie,
  • rejestracja,
  • wyszukiwarka,
  • formularz kontaktowy,
  • ankiety,
  • składanie zamówień.

Każdy formularz powinien znajdować się wewnątrz znacznika: <form>
</form>

label
Opisuje pole formularza. <label for="email">
Adres e-mail </label>
