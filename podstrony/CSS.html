1. Czym jest CSS?

CSS (Cascading Style Sheets) odpowiada za wygląd strony.

Pozwala określać:

  • kolory,
  • rozmiary,
  • odstępy,
  • układ,
  • animacje,
  • responsywność.

2. Dodawanie CSS do dokumentu HTML

Internal
W sekcji <head> dokumentu HTML, umieszczamy kod CSS. <style>
p{
color:red;
}
</style>

Inline
Bezpośrednio w atrybucie style elementu HTML piszemy CSS. <p style="color:red;"> Tekst </p>

External
Tworzymy osobny plik z rozszerzeniem .css, czyli arkusz stylów, w którym umieszczamy kod CSS. Następnie w sekcji <head> dokumentu HTML, dodajemy link do tego pliku.
HTML: <link rel="stylesheet" href="style.css">
CSS:
p{
color:red;
}

Najczęściej stosowana metoda to zewnętrzny arkusz stylów

6. Model pudełkowy

Każdy element strony składa się z:

Margin
Odstęp zewnętrzny. margin:20px;

Padding
Odstęp wewnętrzny. padding:20px;

Border
Obramowanie. border:2px solid black;

Content
Zawartość elementu. width:200px;

7. Jednostki

px - Piksele. width:300px;

% - Procent rodzica. width:50%;

em - Jednostka zależna od rozmiaru czcionki elementu. width:20em;

rem - Jednostka zależna od rozmiaru czcionki dokumentu. width:20rem;

vw - Procent szerokości okna. width:50vw;

vh - Procent wysokości okna. height:50vh;

10. Flexbox

System układania elementów w jednym wymiarze. .container{ display:flex; }

Najważniejsze właściwości:

align align-items:center; align-content:center; align-self:center;

justify-content justify-content:center;

flex-direction flex-direction:row;

gap gap:10px;

flex-wrap flex-wrap:wrap;

Idealny do: menu, kart, formularzy, centrowania elementów.

11. Grid

System układania elementów w dwóch wymiarach.

.grid{ display:grid; }

Tworzenie kolumn
grid-template-columns:repeat(3,1fr);

Tworzenie wierszy
grid-template-rows:repeat(2,100px);

Tworzenie odstępów między kolumnami i wierszami gap:10px;

Tworzenie obszarów
grid-template-areas:
"header header header"
"sidebar content content"
"footer footer footer";

Umieszczanie elementów
.header{grid-area:header;}
.sidebar{grid-area:sidebar;}
.content{grid-area:content;}
.footer{grid-area:footer;}

Idealny do budowy całych stron.

12. Pseudoklasy

:hover
Po najechaniu kursorem. button:hover{
background:red;
}

:active
Podczas kliknięcia. button:active{
transform:scale(0.95);
}

:focus
Po zaznaczeniu pola. input:focus{
border:2px solid blue;
}

:first-child
Pierwszy element. li:first-child{
color:red;
}

:last-child
Ostatni element. li:last-child{
color:red;
}

:nth-child() Wybrany element. li:nth-child(3){
color:red;
}

13. Pseudoelementy

::before
Dodaje zawartość przed elementem. p::before{
content:"★ ";
}

::after
Dodaje zawartość po elemencie. p::after{
content:"!";
}

14. Transformacje

Przesuwanie transform:translateX(50px);

Obrót transform:rotate(45deg);

Skalowanie transform:scale(1.2);

15. Przejścia (Transitions)

Płynna zmiana właściwości. button{ transition:0.3s; } Najczęściej używane z: hover, active, focus.

16. Animacje

Tworzenie animacji: @keyframes ruch{

from{
left:0;
}

to{
left:300px;
}
}

Uruchomienie:
.box{
animation:ruch 3s;
}

Najważniejsze właściwości:

animation-name
Wskazuje na stworzoną animację animation-name: animacja;

animation-duration
Określa czas trwania animacji animation-duration: 5s;

animation-delay
Opóźnia animację o wskazany czas animation-delay: 10s;

animation-iteration-count
Mówi ile razy ma się powtórzyć animacja animation-iteration-count: 2;

animation-fill-mode
Pozwala elementowi zachować właściwości z ostatniego kroku animacji animation-fill-mode: forwards;

Wszystkie te właściwości da się uzyskać za pomocą animation: ;

17. Responsywność

Responsywność oznacza dostosowanie strony do:
  • telefonów,
  • tabletów,
  • laptopów,
  • monitorów.

Najczęściej używane jednostki: %, vw, vh, rem,

@media (min-width:489px;)and(max-width:1000px;){

font-size:20px;

}

@media
Oznacza formatowanie responsywne

(min-width: ) i (max-width: )
Zaznaczają przedział szerokości ekranu, dla którego ma wystąpić to formatowanie

"and" służy jako spójnik dwóch granic - górnej i dolnej, ale można ustalić samą granicę dolną lub górną.
